# Bank Marketing Data Transformation

## Overview
The Bank Marketing Data Transformation project focuses on cleaning, structuring, and preparing client data collected by a bank during a recent marketing campaign for personal loans. The objective is to ensure the data conforms to specified structures and data types for seamless integration into a PostgreSQL database.

## Project Contents
This project cleans and reformats the raw data into three separate CSV files, specifically designed for improved data analysis:

- `client.csv`: Contains demographic and account-related details of clients.
- `campaign.csv`: Captures interaction metrics and outcomes from marketing campaigns.
- `economics.csv`: Provides relevant economic data points pertinent to the clients.

## Features
- Data cleaning and transformation based on specified requirements
- Structured datasets for enhanced usability in PostgreSQL
- Date management for effective time-series tracking

## Technologies Used
- Python
- Pandas
- NumPy
